# react-native-dual-picker

A highly customizable React Native dual value picker supporting single, range, and paired value selection with smooth scrolling UI.

## Installation


```sh
npm install react-native-dual-picker
```


## Usage


```js
import { multiply } from 'react-native-dual-picker';

// ...

const result = multiply(3, 7);
```
